Registering an Application

So you're registered now, and you have identified particular products - paid API subscriptions - you would like to consume, you are now ready to start consuming our API's. To register an application on your developer account to start consuming our API's, use the form here to register a new application. Please Ensure you are logged in

  1. Click the dropdown under your Logged in name
  2. CLick "My Profile"
  3. You are directed to a Profile page
  4. Register new application properties by clicking the Register application
  5. Fill out the details properly and upload an image of size 100 x 100 pixels
  6. Save the application properties for review
